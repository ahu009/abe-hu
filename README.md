# My Personal Website
##### -- React, Redux, Webpack

Link: abehu.me

### How to Run
- Fork or Clone Repository
- cd into repo
- Run one of the two commands below:

For Yarn:
```
yarn install;
yarn run webpack;
```

For npm:
```
npm install;
npm run webpack;
```

- App will run on port: http://127.0.0.1:8888/
